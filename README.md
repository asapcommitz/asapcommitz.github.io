## Mentality

I'm tired of corporate minimalism and the "Software as a Subscription" model. I want my tools to be:
1. **Lightweight**: No 2MB bundles for 200 words of text.
2. **Transparent**: Open source by default.
3. **Fun**: Because why build things if they aren't cool?
